# Open-LMX Open Core Model

## Principle

Open-LMX follows an open core model.

The open source foundation defines the common operating model, vocabulary, schemas, data models, governance principles, interoperability contracts, and reference standards.

The commercial platform delivers managed services, hosted infrastructure, enterprise workflows, integrations, analytics, verification, support, and operational guarantees on top of the foundation.

## Open Source Core

Repository:

```text
open-lmx/foundation
```

The open source foundation includes:

- Common operating model
- Canonical vocabulary
- Data models
- JSON schemas
- Governance principles
- Trust model
- Learning models
- Knowledge models
- Assessment models
- Credential models
- Reference examples
- Interoperability contracts

The foundation should remain broadly usable by institutions, employers, governments, open source projects, researchers, and ecosystem partners.

## Commercial Platform

Repository:

```text
open-lmx/platform
```

The commercial platform includes:

- Hosted registry
- Managed knowledge graph
- Learner skill passport
- Institution portal
- Employer portal
- Recruiter portal
- Credential verification service
- Assessment operations
- Skill-gap analytics
- Job and role matching
- Career progression intelligence
- Enterprise integrations
- Governance workflows
- Audit and compliance reporting
- Support and managed operations

## Boundary Rule

Foundation defines standards.
Platform delivers outcomes.

Foundation should not depend on commercial platform code.
Platform may depend on foundation models and schemas.

## Commercial Value

The platform creates value through:

- Network effects
- Verification workflows
- Managed trust infrastructure
- Enterprise integrations
- Analytics and reporting
- Operational support
- Compliance features
- Multi-tenant services
- Ecosystem marketplace

## Adoption Strategy

Open source foundation adoption increases trust, interoperability, and ecosystem reach.

Commercial platform adoption monetizes enterprise operations, hosted workflows, verification, analytics, and managed services.

## Strategic Positioning

Open-LMX Foundation is the open standard for education-to-employment interoperability.

Open-LMX Platform is the commercial operating network built on that standard.
